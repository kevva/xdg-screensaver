# xdg-screensaver

> Start the screensaver


## Install

```
$ npm install --save xdg-screensaver
```


## Usage

```js
const xdgScreensaver = require('xdg-screensaver');

xdgScreensaver().then(() => {
	console.log('Screensaver started');
});
```


## API

### xdgScreensaver()

Returns a promise that resolves nothing.
